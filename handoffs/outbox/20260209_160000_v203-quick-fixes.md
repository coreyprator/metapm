## Fix 1: UAT Submit Pending Status

**Problem**: Backend rejected `status: "pending"` with 422 error

**Root Cause**: `UATStatus` enum only allowed `passed` | `failed`

**Fix**: Added `PENDING` to enum in `app/schemas/mcp.py`:
```python
class UATStatus(str, Enum):
    PASSED = "passed"
    FAILED = "failed"
    PENDING = "pending"  # NEW
```
