### 1. CRITICAL: UAT Submit 422 Error ‚úÖ

**Root Cause:** `UATDirectSubmit` schema missing `tested_by` field

```python

```

### 2. "All" Project Filter Broken ‚úÖ

**Root Cause:** Filter bar rebuilt via innerHTML but "All" button onclick handler lost

**Fix:** Changed `buildFilters()` in `roadmap.html` to create "All" button with `document.createElement()` and attach onclick properly:
```javascript

```

### 3. Missing Projects (EM, SF, PM) ‚úÖ
